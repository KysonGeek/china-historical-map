# 中国历史疆域图

交互式网页:沿年份时间轴逐段浏览各政权疆域的示意性划分,含明暗两套主题。

**在线预览**:https://kysongeek.github.io/china-historical-map/

> ⚠️ 疆域为示意性近似(按现代省份组合划分),**非严格史实边界**。

## 功能

- 🗺️ 以现代省级行政区为底图,可开关省界显示
- 🕰️ 年份时间轴滑块 + 时期快捷按钮:22 个年份断面,从前221年到现代;
  三国(汉末群雄 → 鼎立 → 蜀亡)与五代十国(后梁 → 后晋 → 后周)等分裂期拆分为多个断面,十国政权逐一并存显示
- 🎨 各政权以不同色块显示,配有政权名与都城标注;同屏最多十政权,配色经色盲区分度与对比度校验
- 🌗 明暗主题一键切换(右上角按钮),选择保存在浏览器本地
- 📋 悬停 / 点触政权区域,信息卡显示:政权名、都城、存续年份、一句话简介
- ✨ 切换断面时色块平滑过渡
- 📱 移动端可用(触摸滑动时间轴、点触查看详情)

## 使用

无需构建,但页面需通过 HTTP 访问(会读取 `territories.json`):

```bash
python3 -m http.server 8000   # 然后访问 http://localhost:8000
```

地图底图与 ECharts 从 CDN 加载,需联网。

## 修改疆域数据

疆域数据全部在 **`territories.json`** 中,与页面代码分离。结构:

```jsonc
{
  "provinces": ["北京", "天津", …],          // 省级行政区简称表
  "palette": { "dark": {…}, "light": {…} },  // 明暗两套主题色,键名如 "amber"
  "snapshots": [                              // 年份断面,按 year 升序(负数为公元前)
    {
      "year": 955,                 // 断面年份
      "era": "五代十国",           // 时期快捷按钮名(相邻同名断面归为一组)
      "label": "后周·显德年间",    // 断面说明(时间轴角标显示)
      "regimes": [
        {
          "name": "后周",                    // 政权名(地图标注用)
          "color": "amber",                  // palette 中的色键,也可直接写 "#RRGGBB"
          "capital": "东京开封府",           // 都城
          "capCoord": [114.31, 34.8],        // 都城坐标 [经, 纬],null 则不画标记
          "years": "951 – 960",              // 存续年份
          "desc": "……",                      // 一句话简介
          "label": [113.8, 36.6],            // 政权名标注位置 [经, 纬]
          "size": 20,                        // 标注字号
          "provinces": ["河南", "山东", …]   // 覆盖的现代省份(简称)
        }
      ]
    }
  ]
}
```

直接增删 `provinces` 里的省份、调整 `label` 坐标或新增断面即可,无需改动页面代码。
同一断面内各政权颜色不要重复,省份不要同属两个政权。

## 技术说明

- [ECharts 5](https://echarts.apache.org/)(map 系列)+ 阿里 DataV 省级 GeoJSON(备用源:echarts@4 china.json),请求带超时自动降级
- 断面切换动画为逐省颜色插值(requestAnimationFrame),尊重 `prefers-reduced-motion`
- 政权名 / 都城标注为 HTML 覆盖层,衬线字体(Noto Serif SC)
- 明暗主题分别使用独立校验过的色板(色盲区分度 ΔE、对比度、亮度带、色度下限)

## 历史口径说明

- 每个断面取该年份前后的大势作示意,如南北朝取约公元 500 年(北魏 / 南齐)、明取永乐、清取乾隆
- 元、清、唐等实际疆域远超现代国界的时期,示意图仅绘制现代国界范围内的部分,实际范围在信息卡中以文字说明
