# 中国历史疆域图

单文件交互式网页:在不同历史时期之间切换,查看各政权疆域的示意性划分。

**在线预览**:https://kysongeek.github.io/china-historical-map/

> ⚠️ 疆域为示意性近似(按现代省份组合划分),**非严格史实边界**。

## 功能

- 🗺️ 以现代省级行政区为底图,可开关省界显示
- 🕰️ 时间轴滑块 + 朝代快捷按钮,覆盖 13 个时期:
  秦 · 西汉 · 三国 · 西晋 · 南北朝 · 隋 · 唐 · 北宋/辽/西夏/大理 · 南宋/金 · 元 · 明 · 清 · 现代
- 🎨 各政权以不同色块显示,配有政权名与都城标注;分裂时期(三国、南北朝、宋辽金)正确并存多个政权
- 📋 悬停 / 点触政权区域,信息卡显示:政权名、都城、存续年份、一句话简介
- ✨ 切换时期时色块平滑过渡
- 📱 移动端可用(触摸滑动时间轴、点触查看详情)

## 使用

无需构建,直接用浏览器打开 `index.html` 即可(依赖 ECharts 与省级 GeoJSON,均从 CDN 加载,需联网)。

## 修改疆域数据

政权 → 现代省份的映射全部硬编码在 `index.html` 顶部的 `PERIODS` 数组中,每个政权包含:

```js
{
  name: '魏',                    // 政权名(地图标注用)
  color: C.amber,                // 色块颜色
  capital: '洛阳',               // 都城
  capCoord: [112.45, 34.62],     // 都城坐标 [经, 纬]
  years: '220 – 266',            // 存续年份
  desc: '……',                    // 一句话简介
  label: [112.5, 37],            // 政权名标注位置 [经, 纬]
  provinces: ['河北', '山西'],   // 覆盖的现代省份(简称)
}
```

直接增删 `provinces` 里的省份、或调整 `label` 坐标即可修正,无需改动其他代码。

## 技术说明

- [ECharts 5](https://echarts.apache.org/)(map 系列)+ 阿里 DataV 省级 GeoJSON(备用源:echarts@4 china.json)
- 时期切换动画为逐省颜色插值(requestAnimationFrame),尊重 `prefers-reduced-motion`
- 政权名 / 都城标注为 HTML 覆盖层,衬线字体(Noto Serif SC)
- 配色在深色底上通过了色盲区分度与对比度校验

## 历史口径说明

- 南北朝以**约公元 500 年**(北魏 / 南齐)为代表断面;唐取盛唐、明取永乐、清取乾隆时期
- 元、清、唐等实际疆域远超现代国界的时期,示意图仅绘制现代国界范围内的部分,实际范围在信息卡中以文字说明
